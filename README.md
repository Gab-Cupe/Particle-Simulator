# Particle-Simulator
Particle Simlator es un motor de visualización y simulación de partículas en tiempo real construido con React y Three.js. Permite a los usuarios definir trayectorias complejas mediante la evaluación dinámica de fórmulas matemáticas (f(t, x, y, z)), visualizando vectores de fuerza y rastros cinéticos en un entorno tridimensional optimizado.
